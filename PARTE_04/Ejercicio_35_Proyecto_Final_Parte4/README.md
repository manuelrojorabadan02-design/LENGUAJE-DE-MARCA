# Ejercicio 35: Proyecto Integrador Parte 4 (Gestor de Películas)

## Objetivo
Agrupar y demostrar competencias reunificadas: HTML Semántico, SASS modular, Gulp (Pipeline completo), XML/XSD y JS moderno.

## Instrucciones
1. Construye una estructura de carpetas de SASS basándote en un patrón (ej. partials para bases, componentes, y layout).
2. Configura un gulpfile.js con las tareas para compilar SCSS, minificar CSS (gulp-clean-css) y automatizar la recarga o al menos observar (watch) los cambios.
3. El proyecto debe ser una SPA de una sola página para listar películas o libros.
4. El repositorio de datos inicial se nutrirá de datos.xml, el cual debe validar perfectamente con schema.xsd.
5. JavaScript debes cargar ese XML utilizando ES6 (ideal si separas módulos usando import/export si montas un builder, o al menos un script limpio y ordenado).
6. Implementa un pequeño filtro de texto con JS: un <input type="text"> que al escribir se oculten del DOM (o se borren las tarjetas) aquellas que no coincidan con el título.
7. ¡Demuestra todo lo aprendido en la PARTE_03 y PARTE_04 en este proyecto final!
